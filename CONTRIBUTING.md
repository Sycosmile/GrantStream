# Contributing to GrantStream

Thanks for your interest in contributing! GrantStream is actively seeking contributors.

## Ways to Contribute
- 🔒 Smart contract development (Solidity escrow logic)
- ⚛️ Frontend components (React + TypeScript)
- 🧪 Tests (Vitest unit + integration)
- 📝 Documentation improvements
- 🐛 Bug reports and fixes

## Getting Started
1. Fork the repository
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Open a PR with a clear description

## Commit Convention
We use [Conventional Commits](https://www.conventionalcommits.org/):
- `feat:` new feature
- `fix:` bug fix
- `chore:` maintenance
- `docs:` documentation
- `test:` adding tests

## Good First Issues
Check issues labeled `good first issue` to get started.

## PR Reviews
All PRs reviewed within 48 hours. Be descriptive — what, why, and how.
